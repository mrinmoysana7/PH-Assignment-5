1️⃣ What is the difference between var, let, and const?

ANS:  In JavaScript, var, let, and const are used to declare variables, but they behave differently.

var: It is the old way of declaring variables. It is function-scoped and can be re-declared and updated. Sometimes it creates confusion because it doesn’t follow block scope.

let: It is block-scoped, which means it only works inside the block {} where it is declared. It can be updated but cannot be re-declared in the same scope.

const: It is also block-scoped, but its value cannot be changed once assigned. It is used for constants.


2️⃣ What is the spread operator (...)?

ANS: The spread operator (...) is used to expand elements of an array or object.

It is useful for copying arrays, merging arrays, or passing values.


3️⃣ What is the difference between map(), filter(), and forEach()?

ANS: These are array methods used to work with array elements.

map(): It creates a new array by applying a function to each element.

filter(): It creates a new array with elements that match a condition.

forEach(): It just loops through the array and performs an action but does not return a new array.


4️⃣ What is an arrow function?

ANS: An arrow function is a shorter way to write functions in JavaScript using =>.

It makes code simpler and more readable.


5️⃣ What are template literals?

ANS: Template literals are used to create strings using backticks (`) instead of quotes.

They allow embedding variables and expressions using ${}.
